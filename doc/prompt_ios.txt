The goal is to implement an iOS Share Extension so that when a user taps:

Share → Stackrl

from apps like Safari, YouTube, Instagram, etc., the extension receives a shared URL and sends it to the main app.

⸻

PROJECT CONTEXT
	•	Expo project already exists
	•	Using Expo Dev Client (not Expo Go)
	•	iOS target only for now
	•	React Native frontend
	•	Backend API already exists
	•	We only need to capture shared URLs

⸻

WHAT I WANT YOU TO BUILD

Create a complete iOS Share Extension implementation that:
	1.	Appears inside the iOS Share Sheet as “Stackrl”
	2.	Accepts shared content of type:
	•	public.url
	•	public.text
	3.	Extracts the URL from the shared item
	4.	Stores the URL temporarily using App Groups shared container
	5.	Opens the main Stackrl app via deep link:
stackrl://import
	6.	Passes the shared URL to the main app
	7.	Clears temporary storage afterward

⸻
