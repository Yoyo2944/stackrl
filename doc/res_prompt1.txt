  Point important

  Avec le SDK 55 canary, utilisez npm install --legacy-peer-deps pour les packages hors Expo (ex : zustand) à cause d'un conflit de peer deps sur react-dom.

  Pour lancer : cd /Users/steven/Dev/stackrl && npx expo start
